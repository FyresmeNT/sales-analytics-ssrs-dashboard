# Разработка дашборда анализа продаж
Интерактивный BI-отчет для анализа продаж, построенный на стеке Microsoft. От сложных SQL-запросов до готового дашборда с визуализацией.

### Основные возможности

- **Анализ продаж** по датам, заказам и категориям товаров
- **Автоматический расчет** выручки и количества товаров
- **Интерактивная фильтрация** по подкатегориям продуктов
- **Визуализация данных** через таблицы и круговые диаграммы
- **Публикация на сервере** SSRS для доступа через браузер

### Технологии

- **Database:** Microsoft SQL Server
- **Query Language:** T-SQL
- **Reporting:** SQL Server Reporting Services (SSRS)
- **Development:** SQL Server Data Tools (SSDT)
- **Sample Data:** AdventureWorks2017


### **Сложная SQL-логика**
- Применение запросов с объединением таблиц для внутренней логики
- Агрегация данных

### **Интерактивный интерфейс**
- Группировка данных: Дата → Заказ → Товар
- Параметры фильтрации: Выбор подкатегорий товаров
- Итоги: Промежуточные и общие автоматические расчеты

### **Визуализация**
- Круговая диаграмма распределения заказов
- Процентное отображение долей категорий
- Профессиональное форматирование (валюта, даты)

### **Развертывание**
- Конфигурация для публикации на SSRS-сервере
- Доступ через веб-интерфейс диспетчера отчетов




## Быстрый старт

1. Восстановите базу данных AdventureWorks2017 на SQL Server

2. Откройте файл SalesOrders.rdl в SQL Server Data Tools

3. Обновите подключение к вашей базе данных

4. Запустите предварительный просмотр для проверки

5. Опубликуйте отчет на вашем SSRS-сервере

Проект демонстрирует полный цикл создания аналитического отчета: от проектирования запросов до публикации готового решения.
